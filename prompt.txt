DISTRICT PORTAL — BUILD PROMPT
NYU Senior Seminar (Fall 2026) · pilot for NY State Senate District 20 (Sen. Zellnor Myrie)
Last revised: 2026-07-15

WHAT THIS IS
A one-page dashboard that collates publicly available information about a single
legislative district, so students can find it fast and then confirm it at the source.
It is a starting point for reporting, never a citation.

GROUND RULES (apply to every pane)
- No invented data. A failed fetch shows an honest gap, never a guess or an interpolation.
- Every pane names its source, the retrieval date, and what could be wrong with it.
- Prefer primary sources (Census API, Board of Elections, NYC Open Data) over aggregators.
- Charts start their y-axis at zero. Category and bracket choices are disclosed.
- The page is a claim, not a fact. It says so, and tells the reader to cite the origin.

WHAT THE PAGE CONTAINS
1. Map of the district — current (2022) lines solid, previous (2012) lines dashed. Census TIGER/Line.
2. Who are residents — population, median household income, share foreign-born, median age,
   each shown against the citywide figure. Source: ACS 5-year (2020-24).
3. Census detail — race/ethnicity and age vs. NYC; population and median income over time,
   with the 2022 redistricting break marked. Source: ACS 5-year via the Census API.
4. Background on the member — Wikipedia, election history with vote totals, signature priorities.
5. Local news — last 6 months across local outlets, deduplicated, tagged by topic, filterable.
6. Resources — primary-source portals for going deeper.

HOW IT'S BUILT
A single Python script (build/build_portal.py) fetches each source, writes one JSON file per
pane (each with a provenance block), and bakes them into data/portal_data.js so the page runs
from a static file or GitHub Pages. index.html is presentation only.
